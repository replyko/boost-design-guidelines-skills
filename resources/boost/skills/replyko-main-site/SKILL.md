# Replyko Main Site

Use this skill for work in `replyko.com`.

## When To Use This Skill

Activate this skill for:
- homepage sections
- pricing and CTA sections
- blog pages
- competitor alternative pages
- comparison pages
- use-case pages
- legal pages
- localized public content
- marketing Blade and Tailwind UI work

## Stack Context

- Laravel
- Blade
- Tailwind CSS
- Alpine
- Filament-backed content/admin flows

## Design Intent

- optimize for conversion, trust, and product clarity
- use Coral only for primary conversion CTAs
- use Indigo for brand framing and structure
- keep layouts spacious and easy to scan
- use Emerald for verified and trust indicators

## Preserve Existing Direction

This repo already aligns closely with the target Replyko brand system.

Default behavior should be to reinforce the current Indigo / Coral direction rather than redesign it.
