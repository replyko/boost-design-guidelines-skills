## Implementation Intent

- Optimize for workflow clarity and speed
- Keep the visual language consistent with the main site
- Use Coral sparingly and only for true priority actions
- Prefer structure, readability, and predictable interaction patterns
- Respect the existing Inertia, React, TypeScript, and shared component architecture while guiding new UI toward the target shared brand system

## Avoid

- Landing-page aesthetics inside the product
- Decorative CTA treatments on normal controls
- One-off component styling that drifts away from the shared system
